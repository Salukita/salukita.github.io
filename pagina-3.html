introducción a JADE

una plataforma software para el desarrollo de agentes, implementada en Java.plataforma JADE soporta la coordinación de múltiples agentesy proporciona una implementación estándar del lenguaje de comunicación FIPA-ACL,facilita la comunicación entre agentes y permite la detección de servicios que se proporcionan en el sistema.fue desarrollado originalmente por Telecom Italia y se distribuye como software libre.
